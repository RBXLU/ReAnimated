# ReAnimated v1.1

## 🇷🇺 Русский

### Что нового
- ✨ **Поддержка NeoForge!** Мод теперь работает на NeoForge для версий **1.21.1, 1.21.3, 1.21.4, 1.21.5, 1.21.8, 1.21.10, 1.21.11** — со всем набором анимаций (заставка, кнопки, меню, контейнеры, подсветка слота).
- ⚙️ **Экран настроек в игре теперь и на NeoForge, и на Minecraft 26.1 / 26.2.** Заходишь в **Настройки → Настройки графики**, жмёшь **«ReAnimated settings»** — ползунки длительности/дистанции/траектории и переключатели для каждой анимации, плюс ссылка на автора.

### Заметки
- На всех загрузчиках настройки также хранятся в `config/reanimated.json` — можно править вручную.
- На Minecraft 26.x экран настроек без заголовка сверху (особенность новой системы рендера), функционально полный.
- NeoForge выпущен не для всех версий: для 1.21.2 / 1.21.6 / 1.21.7 / 1.21.9 NeoForge не существует — там доступен только Fabric.
- Fabric-сборки совместимы и с **Quilt** (нужен Quilted Fabric API). На 26.2 Quilt пока отстаёт по версии загрузчика.

### Совместимость
| Загрузчик | Версии |
|---|---|
| Fabric / Quilt | 1.21.1 – 1.21.11, 26.1, 26.2 |
| NeoForge | 1.21.1, 1.21.3, 1.21.4, 1.21.5, 1.21.8, 1.21.10, 1.21.11 |

## 🇬🇧 English

### What's new
- ✨ **NeoForge support!** ReAnimated now runs on NeoForge for **1.21.1, 1.21.3, 1.21.4, 1.21.5, 1.21.8, 1.21.10, 1.21.11** — with the full set of animations (logo, buttons, menus, containers, slot highlight).
- ⚙️ **In-game settings screen now on NeoForge and Minecraft 26.1 / 26.2 too.** Go to **Options → Video Settings**, click **"ReAnimated settings"** — sliders for duration/distance/easing and toggles for every animation, plus a link to the author.

### Notes
- On every loader, settings are also stored in `config/reanimated.json` for manual editing.
- On Minecraft 26.x the settings screen has no top title (a quirk of the new render system); it is fully functional.
- NeoForge was never released for 1.21.2 / 1.21.6 / 1.21.7 / 1.21.9 — those are Fabric-only.
- Fabric builds also work on **Quilt** (requires Quilted Fabric API). On 26.2 Quilt's loader is currently behind.

### Compatibility
| Loader | Versions |
|---|---|
| Fabric / Quilt | 1.21.1 – 1.21.11, 26.1, 26.2 |
| NeoForge | 1.21.1, 1.21.3, 1.21.4, 1.21.5, 1.21.8, 1.21.10, 1.21.11 |
